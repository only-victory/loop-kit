# /blog : 블로그 글 작성

검색에 걸리고 끝까지 읽히는 **블로그 글**을 쓴다. 산출물은 제목·소제목·본문·메타가 갖춰진, 그대로 발행할 수 있는 마크다운이다.

> 이 스킬의 목표는 "분량을 채우는 것"이 아니라 **"첫 문장으로 붙잡고, 다 읽은 독자가 무언가 얻었다고 느끼게 하는 것"**이다.

## 제1원칙 : 첫 문장이 전부다

도입을 "요즘 ~가 화제입니다"로 시작하지 않는다. **첫 2\~3문장에서 독자의 문제나 호기심을 정확히 건드린다.** 검색해 들어온 독자는 3초 안에 "여기 답이 있다"를 느껴야 머문다.

## 절차 (이 순서를 지킨다)

1. **인터뷰** : 아래 [깊은 인터뷰]로 결·독자·키워드를 끌어낸다.
2. **한 줄 약속** : 글을 쓰기 전에 "이 글을 읽으면 [독자]는 [무엇]을 얻는다"를 한 줄로.
3. **제목·소제목 설계** : 검색 키워드를 자연스럽게 담은 제목(H1) + 소제목(H2) 골격.
4. **작성** : 훅 도입 → 소제목별 본문 → 결론·CTA → 메타(요약·태그).
5. **검증** : `scripts/verify.sh`로 제목·소제목·메타·길이·구두점을 확인한다.

## 깊은 인터뷰 (옵션은 고르게, 내용은 받는다)

**0단계 : 블로그 결 (선택 창으로 고르게 한다)**

- ① 정보·하우투결 : 문제 → 단계별 해결 → 예시 (검색 유입·실용)
- ② 경험·에세이결 : 장면 → 깨달음 → 일반화 (공감·구독)
- ③ 큐레이션·리스트결 : 기준 → N개 항목 → 비교·추천 (스캔·공유)
- ④ 의견·주장결 : 통념 → 반론·근거 → 결론 (토론·확산)

**1단계 : 독자·목적**

- **독자** : 입문자 / 실무자 / 의사결정자 / 일반 대중
- **검색 키워드** : 독자가 검색할 단어 (제목·소제목·첫 문단에 자연스럽게)
- 다 읽은 독자가 **무엇을 하길** 바라나 (구독·댓글·공유·전환)

**2단계 : 핵심 내용**

- **한 줄 약속** : 이 글의 가치
- **핵심 메시지 3개** : 소제목이 될 기둥
- **근거·예시** : 주장마다 사례·데이터·경험

## 블로그 결 골격

| 결 | 구조 |
|----|------|
| 정보·하우투결 | 훅(문제 공감) → 왜 중요 → 단계 1·2·3 → 흔한 실수 → 요약·다음 행동 |
| 경험·에세이결 | 장면(구체) → 전환·갈등 → 깨달음 → 독자에게 주는 의미 |
| 큐레이션·리스트결 | 선정 기준 → 항목 N개(각 한 단락) → 비교 표 → 추천 |
| 의견·주장결 | 통념 제시 → 왜 틀렸나(근거) → 대안 주장 → 반론 수용 → 결론 |

**공통 규약:**
- 제목은 `# `(H1) 하나, 소제목은 `## `(H2) 2개 이상
- 한 문단 3\~5줄, 소제목마다 핵심 한 줄을 먼저
- 글 끝에 메타 : 한 줄 요약 + 태그(검색어) 3\~6개
- em-dash·en-dash 금지, 구분자는 콜론(`:`)

## 출력 형식

```
# 제목 (키워드 포함, 궁금하게)

(훅 도입 2\~3문장 : 독자 문제·호기심)

## 소제목 1
(핵심 한 줄 → 설명 → 예시)

## 소제목 2
...

## 마무리
(요점 정리 + 독자가 할 다음 행동)

---
요약: (한 줄)
태그: 키워드1, 키워드2, 키워드3
```

## 안티패턴 (이게 보이면 다시 쓴다)

- "요즘 ~가 뜨고 있습니다"로 시작 → 첫 문장에서 독자 문제를 건드려라.
- 소제목 없이 긴 글 → 스캔이 안 된다. H2로 끊어라.
- 주장만 있고 예시·근거 없음 → 사례·데이터·경험을 붙여라.
- 결론이 "이상입니다" → 독자가 할 다음 행동으로 끝내라.
- 키워드를 부자연스럽게 반복 → 검색용 스터핑은 역효과. 자연스럽게.
- 메타(요약·태그) 없음 → 검색·공유에서 손해.

## 검증 (필수 : loop-kit 게이트)

```bash
bash ${CLAUDE_SKILL_DIR}/scripts/verify.sh <만든-글.md>
```

- [ ] (자동·HARD) em-dash/en-dash 0건
- [ ] (자동·HARD) 제목 `#`(H1) 존재
- [ ] (자동·WARN) 소제목 `##`(H2) 2개 이상
- [ ] (자동·WARN) 메타(요약·태그) 존재
- [ ] (자동·WARN) 분량 하한(너무 짧지 않음)
- [ ] (육안) 첫 2\~3문장이 독자를 붙잡는가
- [ ] (육안) 주장마다 근거·예시가 있는가
- [ ] (육안) 마지막에 다음 행동이 있는가
