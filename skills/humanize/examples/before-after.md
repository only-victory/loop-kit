# humanize 예시 : before → after

## Before (AI 슬롭)

> 이번 글에서는 원격 근무의 장점에 대해 알아보겠습니다. 원격 근무의 중요성은
> 아무리 강조해도 지나치지 않습니다. 원격 근무는 다음과 같은 혁신적인 이점을
> 제공합니다: 첫째, 생산성 향상. 둘째, 비용 절감. 결론적으로, 원격 근무는
> 체계적으로 도입할 가치가 있다고 할 수 있습니다.

`verify.sh` 결과 : ✗ 금지 패턴 7건 (대해 알아보 / 아무리 강조해도 / 혁신적인 / 다음과 같습니다: / 결론적으로 / 체계적으로 / 라고 할 수 있습니다)

## After (humanized)

> 원격 근무는 두 가지를 바꿨다. 출퇴근에 쓰던 두 시간이 일하는 시간으로
> 돌아왔고, 사무실 임대료가 사라졌다. 다만 효과는 도입 방식에 달렸다.
> 규칙 없이 자율에만 맡기면 협업이 끊긴다.

`verify.sh` 결과 : ✓ 금지 패턴 0건 : humanize 통과

## 무엇이 달라졌나

- 의례적 도입("~에 대해 알아보겠습니다") 삭제, 본론을 첫 문장에 배치
- 추상적 나열("생산성 향상·비용 절감") → 구체("두 시간"·"임대료")
- 빈 수식어("혁신적인")·헤지("~라고 할 수 있습니다") 제거
- 장점만 늘어놓던 글에 한계("협업이 끊긴다")를 더해 사람의 판단을 드러냄
