# 디자인 결 : 브랜드 인상을 고른다

인터뷰 0단계에서 아래 **디자인 결**을 선택 창으로 제시하고, 고른 결의 템플릿을 복제한다. 자유 입력이 아니라 **고르게** 한다.

기존엔 테마 4종을 `:root`만 바꿔 적용했는데, 그건 색만 달랐고 폰트·여백·도형 언어는 같았다. 그래서 색을 **결 안으로 흡수**했다. 결은 색만이 아니라 **폰트·타이포 스케일·여백·도형·CTA 스타일·섹션 분위기**까지 통째로 정의한다. 결이 다르면 같은 내용도 전혀 다른 교육 브랜드가 된다.

---

## 결 4종

### ① 프리미엄 다크
`templates/promo.premium-dark.html.txt`

| 축 | 값 |
|----|----|
| **폰트** | Noto Serif KR(헤드라인·이탤릭) + Pretendard(본문) + JetBrains Mono(레이블) |
| **색** | 깊은 다크 `#09090f`, 크림 글자 `#ede9df`, 금 포인트 `#c8963b`, 올리브 `#7d7c54` |
| **여백** | 넓고 호흡이 긴 여백 (section padding 96px+) |
| **도형** | 헤어라인 구획선, 미니멀 라운드 4px, 세리프 인용 블록 |
| **CTA 스타일** | 골드 테두리 버튼, 호버 시 반전 |
| **언제** | 고급 브랜드 교육, 심화 과정, 마스터클래스, 유명 강사 기반 |

## 결 선택 휴리스틱

- 프리미엄·유명 강사·고가 과정 → **프리미엄 다크**
- 기업·테크·데이터·개발 과정 → **모던 클린**
- 빠른 전환·마감 임박·집중 과정 → **에너지 임팩트**
- 학원·대학·정규·자기계발 → **소프트 아카데믹**

## 다른 결로 바꾸려면

해당 결 템플릿을 복제한다. 내용(섹션 구조·클래스명)은 동일하고, `<style>`과 폰트 `<link>`만 다르다. `verify.sh`를 반드시 통과시킨다.
