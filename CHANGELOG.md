# Changelog

이 프로젝트의 모든 주요 변경을 기록한다. [SemVer](https://semver.org/lang/ko/)를 따른다.

## [0.7.0] : 2026-06-25

### 개편 (slide 디자인 결 시스템)

- 색 테마 5종(`:root` 교체)을 폰트·그리드·도형·모티프를 통째로 정의하는 **디자인 결 4종**으로 대체
  - 에디토리얼 : Fraunces 세리프·비대칭 그리드·금/세이지 (다크)
  - 스위스 그리드 : Archivo 그로테스크·강한 그리드·시그널 레드 (유일한 라이트)
  - 레트로 터미널 : JetBrains Mono·네온·스캔라인, edu 팔레트 흡수
  - 소프트 글래스 : 인디고 그라데이션·글래스 카드·큰 라운드
- `reference/themes.md` 삭제 → `reference/design-languages.md`로 대체
- `.lead`/`.body`에 measure(54~62ch) 적용 : 좌측쏠림 올바르게 해결
- 우상단 폴리오(결 이름) 추가 : 우측 시각 균형
- `outro` 레이아웃 신규 : 행동 촉구 전용
- `verify.sh` WARN 추가 : `✦`·keydown·prefers-reduced-motion
- 4개 결 템플릿 모두 HARD 위반 0건 통과 확인

## [0.6.0] : 2026-06-25

### 추가 (promo 스킬)

- `promo` 스킬 신규 : 교육 프로그램 홍보 웹페이지를 인터뷰 기반으로 생성
- 인터뷰 3단계 : 테마·유형 선택 → 핵심정보 → 콘텐츠 → 차별화
- 테마 4종 : 다크 골드·라이트 블루·라이트 크림·네이비 블루
- `verify.sh` : h1·신청 버튼·footer 요소·em-dash·외부 JS CDN·허용 CSS CDN 검사
- README 설치 3단계 추가 : `/reload-plugins` 필수 명시

## [0.5.0] : 2026-06-25

### 심화 (report 레퍼런스 품질로 — 킬러 3종 모두 심화 완료)

- `report` 인터뷰화 : 보고서 종류·독자·밀도를 선택 창으로, 내용은 직접 입력
- `reference/structures.md` : 역피라미드(결론 먼저) + 종류별 골격 + 독자별 깊이 + 수치 다루는 법
- `templates/report.html.txt` : 완성형 베이스 (인라인 CSS·인쇄 친화·자체완결, 게이트 통과)
- `verify.sh` 강화 : 제목 `<h1>`·`@media print` 검사 추가
- 템플릿 통과·위반 fixture 실패 검증 완료

## [0.4.0] : 2026-06-25

### 심화 (humanize 레퍼런스 품질로)

- `humanize` 인터뷰화 : 문서 종류·목표 어조·강도를 선택 창으로, 원문은 직접 입력
- `reference/patterns.md` : 슬롭 패턴 10개 카테고리 카탈로그 (심각도 S1/S2/S3 + before→after)
- `verify.sh` 강화 : 심각도 집계 + **A~D 등급** + **변경률 가드**(원문 대비 30/50% 초과 경고). S1 잔존 시 실패
- 등급 D/C/A fixture로 동작 검증 (D 실패·C·A 통과·변경률 측정)

## [0.3.2] : 2026-06-25

### 수정 (slide 정렬 일관성)

- 슬라이드 세로 중앙 정렬 → **고정 프레임**(`.frame`, 좌측 거터·제목 높이 고정)으로 교체. 장마다 제목이 위아래로 흔들리던 문제 해결
- `.eyebrow`/`.title`/`.lead`/`.body`/`.cols`/`.card` 레이아웃 클래스 정의 → 슬라이드별 임의 위치잡기 금지
- SKILL.md에 "레이아웃 일관성(필수)" 규칙 명시

## [0.3.1] : 2026-06-25

### 개선

- `slide` 인터뷰 : 선택지 질문(색·톤·밀도·목적·청중 수준)은 **반드시 선택 창으로 띄우게** 명시. 텍스트 나열로 서술 유도 금지. 직접 입력은 실제 주제·내용에 한정.
